🔢 NumPy Array Basics in Python

This project demonstrates basic NumPy operations such as creating arrays, performing element-wise operations, and calculating the sum of array elements.

🚀 Features

Create a NumPy array

Perform element-wise addition

Calculate the sum of array elements

Beginner-friendly example

🛠️ Technologies Used

Python 3

NumPy

📦 Installation

Install NumPy using pip:

pip install numpy

▶️ How to Run

Save the code in a file, for example:

numpy_basics.py


Run the script:

python numpy_basics.py

📄 Code Explanation

arr = np.array([1,2,3,4,5])

Creates a NumPy array.

arr_plus_one = arr + 1

Adds 1 to every element using vectorized operations.

np.sum(arr)

Calculates the total sum of the array elements.

[1 2 3 4 5]
[2 3 4 5 6]
15
